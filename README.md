# لوحة مؤشرات الصيانة — قالب عربي

هذا قالب مبسط لستريملايت بالعربية لعرض مؤشرات صيانة أساسية (KPIs) ورسوم بيانية شائعة.

## المتطلبات
- Python 3.10+
- تثبيت المكتبات:

```bash
pip install -r requirements.txt
```

## التشغيل

```bash
streamlit run "maximo_streamlit/maintenance_dashboard_template.py"
```

## الإعدادات (Secrets/Environment)
- `DASH_DATA_URL`: رابط مباشر لملف CSV/Excel (SharePoint/OneDrive/Google Sheets CSV export). في حال عدم ضبطه سيُستخدم `MMC_DATA_URL` إن وُجد. عند الفشل تُستخدم بيانات تجريبية.
- `DASH_TZ`: المنطقة الزمنية (افتراضي `Asia/Riyadh`).
- `DASH_LEFT_LOGO` و `DASH_RIGHT_LOGO`: روابط صور للشعارات (اختياري).

> ملاحظة: يدعم القالب كشف أسماء الأعمدة باللغتين (العربية/الإنجليزية) لبعض الحقول الشائعة مثل: الحالة، الموقع، المنسق، التواريخ (تاريخ البلاغ/الهدف/الإغلاق)، الفئة، النوع، الأولوية.

## المكونات
- مؤشرات: إجمالي الأوامر، المفتوحة، المنجزة (COMP)، المغلقة (CLOSE)، نسبة الإنجاز، الالتزام بالوقت، معدل التراكم، إعادة العمل، متوسط مدة الإنجاز، نسبة PM من (PM+CM).
- رسوم: حسب الموقع، الحالة (دونات)، الزمن (شهري)، المنسق، الفئة، الأولوية، النوع.

## تخصيص
- عدّل قوائم الأسماء المكافئة للأعمدة أو أكواد الحالات بما يلائم نظامك.
- أضف/أزل الرسوم أو المؤشرات حسب الحاجة.
